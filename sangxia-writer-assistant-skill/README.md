# 桑夏作家助手 (Sangxia Writer Assistant)

> 发布前 1 分钟检查，避免发布后 1 小时调试。
>
> 你的专属 AI 写作助手，模仿桑夏风格创作高质量技术文章。

---

## 🎯 这是什么？

**桑夏作家助手** 是一个 OpenClaw Skill，专为模仿桑夏博客写作风格而设计。

**核心能力**：
- ✍️ **风格模仿** - 自动使用桑夏写作模板
- 🔍 **素材搜集** - 搜索本地和网络相关资料
- 📝 **文章创作** - 根据主题创作完整文章
- 🎨 **配图建议** - 分析文章内容建议配图位置
- ✅ **风格检查** - 自动检查文章是否符合风格规范

---

## 🚀 快速开始

### 安装

Skill 位于：
```
~/.openclaw/workspace/skills/sangxia-writer-assistant/
```

### 使用

**1. 搜集素材**：
```bash
cd ~/.openclaw/workspace/skills/sangxia-writer-assistant
node scripts/search-materials.js --topic "AI Agent" --output materials.md
```

**2. 创作文章**：

对 AI 说：
```
帮我写一篇关于 "AI Agent" 的文章，使用桑夏写作风格
```

AI 会自动：
- 加载写作模板
- 参考搜集的素材
- 按照桑夏风格创作

**3. 风格检查**：
```bash
node scripts/check-style.js articles/新文章.md
```

---

## 📋 检查项详解

### 风格检查 (check-style.js)

| 检查项 | 说明 | 修复建议 |
|--------|------|----------|
| **标题格式** | 是否有副标题 | 使用"主标题：副标题"格式 |
| **开场引用** | 是否有引用块 | 用引用块提出核心问题 |
| **场景对比** | 是否有前后对比 | 添加"没有 X"vs"有 X"场景 |
| **类比解释** | 是否有 3 个类比 | 用表格形式提供类比 |
| **表格使用** | 是否有表格 | 在对比/列表时使用表格 |
| **架构图** | 是否有 ASCII 图 | 用框图展示工作原理 |
| **检查清单** | 是否有 checklist | 添加可操作的检查项 |
| **系列导航** | 是否有导航表 | 添加系列文章导航 |
| **字数统计** | 是否有统计 | 添加"*字数：约 X 字*" |
| **Frontmatter** | 是否完整 | 检查 title/date/category/tags |
| **超长行检查** | 每行<100 字符 | 拆分长行或使用多行字符串 |
| **代码块闭合** | ```是否成对 | 检查代码块标记 |

## 🛠️ 工具使用

### 素材搜索工具

**用途**：搜索与主题相关的资料

**命令**：
```bash
node scripts/search-materials.js --topic "<主题>" [选项]
```

**选项**：
| 选项 | 说明 |
|------|------|
| `--topic <主题>` | 搜索主题（必需） |
| `--output <文件>` | 输出文件（默认：materials.md） |
| `--local-only` | 只搜索本地文章 |

**示例**：
```bash
# 搜索 "Skills" 相关资料
node scripts/search-materials.js --topic "Skills"

# 搜索 "AI Agent" 并指定输出文件
node scripts/search-materials.js --topic "AI Agent" --output ai-materials.md

# 只搜索本地文章
node scripts/search-materials.js --topic "Harness" --local-only
```

**输出**：
- 本地相关文章列表
- 网络参考资料
- 写作建议
- 配图建议
