# Sangshy Skills

Claude Code 全栈研发 Skills 集合，覆盖从需求到测试的完整研发链路。

## Skills

| Skill | 说明 |
|-------|------|
| [requirement-analysis](./requirement-analysis/) | 产品经理 Skill，将模糊需求转化为结构化 PRD 文档，包含需求澄清、竞品分析、用户故事和验收标准 |
| [architecture-design](./architecture-design/) | 架构师 Skill，基于 PRD 输出完整技术方案，涵盖技术选型、接口设计、数据库设计和架构图 |
| [code-implementation](./code-implementation/) | 全栈开发 Skill，根据架构方案输出可运行代码，支持 React/Vue 前端、Go/Java/Node/Python 后端、小程序和 Tauri |
| [code-review](./code-review/) | 代码审核 Skill，对代码进行全面的质量/安全/规范/性能审查，输出结构化 Review 报告 |
| [testing](./testing-web/) | 测试验证 Skill，基于需求和设计生成测试用例并执行自动化测试，覆盖 API、E2E、性能和边界值测试 |

## 安装

```bash
git clone https://github.com/sangshy-go/sangshy-skills.git
cp -r sangshy-skills/*/ ~/.claude/skills/
```

或将整个仓库作为 skills 目录引用。
